# income_simulator

소득 & 재테크 통합 시뮬레이터 - 커리어 경로별 소득 비교 및 재무 계획 도구

## 소개

IT 직군과 공무원/군인/경찰/소방 등 공공 직군의 소득을 비교하고, 투자·절세·FIRE 은퇴 계획까지 종합적으로 시뮬레이션할 수 있는 웹 애플리케이션입니다.

## 주요 기능

| 구분 | 기능 | 설명 |
|------|------|------|
| 소득 분석 | 소득 비교 | IT vs 공공직군 누적 소득 비교 |
| | IT 연봉 통계 | KOSA 공식 17개 직무별 연봉 분포 |
| | 실수령액 계산 | 4대 보험 + 소득세 공제 후 계산 |
| 목표 설정 | 목표 역산 | 목표 누적소득 달성을 위한 필요 연봉 계산 |
| | 추월 시점 | 특정 직군 소득 추월 시점 분석 |
| | 마일스톤 | 1억/3억/5억/10억 달성 타임라인 |
| 투자/재테크 | FIRE 은퇴 | 몬테카를로 시뮬레이션 기반 조기 은퇴 계획 |
| | 투자 시뮬레이션 | 주식/채권/현금 포트폴리오 성장 예측 |
| | 복리 효과 | 원금 vs 이자 시각화 |
| 부동산/세금 | 주거비 비교 | 전세/월세/매매 비용 비교 |
| | 연말정산 | 카드·의료비·교육비 공제 계산 |
| | 총 자산 | 소득 + 투자 합산 순자산 추이 |

## 기술 스택

- **Backend**: Python / Flask
- **Frontend**: HTML5 / CSS3 / Vanilla JavaScript
- **차트**: Chart.js 4.4.7
- **스타일**: 다크 테마 (CSS 커스텀 프로퍼티)

## 실행 방법

### 정적 파일로 실행 (서버 불필요)

```bash
open index.html
```

### Flask 서버로 실행

```bash
pip install flask
python app.py
# http://localhost:9847 자동 실행
```
