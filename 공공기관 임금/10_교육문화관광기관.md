# 한국 교육/문화/관광 분야 공공기관 연봉 및 성과보수 종합 분석

> 취업준비생 관점 | ALIO 2024~2025년 기준

---

## 1. 공통 직급체계

대부분의 교육/문화/관광 분야 공공기관은 **5단계 직급체계**를 운용한다:

| 직급 | 일반 호칭 | 승진 소요 최저연수 | 비고 |
|------|-----------|-------------------|------|
| 5급 | 사원/주임 | 신입 입사 | 신규채용 기준 직급 |
| 4급 | 대리 | 2~3년 | 실무 담당 |
| 3급 | 과장 | 3~4년 | 팀원 관리 |
| 2급 | 차장/부장 | 3~4년 | 부서 관리 |
| 1급 | 부장/처장 | 3~4년 | 임원 직전 |

## 3. 직무별 연봉 차이

### 3.1 EBS 직군 구분

| 직군 | 업무 | 연봉 특이사항 |
|------|------|-------------|
| **PD직** | 프로그램 기획/제작 | 야근/특근/출장 수당으로 실수령 높음 |
| **방송기술직** | 촬영, 음향, 조명, 송출 | PD직과 유사한 수당 구조 |
| **ICT기술직/전산직** | ERP 운영, 시스템 개발 | 상대적으로 정시 퇴근 비율 높음 |
| **사무직(경영/행정)** | 기본 호봉 체계 적용 | 기본급 기준 |

> 직군 간 기본급 차이는 크지 않으나, **PD직과 기술직은 특수근무수당**이 추가되어 연간 500~1,000만원 이상 차이 가능

### 3.2 연구기관 (교육개발원, 직업능력연구원, 교육과정평가원)

| 직군 | 초봉 차이 | 비고 |
|------|----------|------|
| **연구직** (석사/박사) | 행정직 대비 +1,000~2,000만원 | 학위 기준 초임 등급 높음 |
| **행정직** | 일반 공공기관 수준 | 기본 호봉체계 |

### 3.3 한국관광공사

| 직무 | 특이사항 |
|------|----------|
| 일반행정직(5급) | 기본연봉 약 3,670만원 + 성과급 별도 |
| 전산직/기술직 | 기본급 동일, IT 자격수당 별도 가능 |
| 해외사무소 근무 | 해외근무수당 추가로 실질 연봉 상승 |

### 3.4 그랜드코리아레저(GKL)

| 직무 | 특이사항 |
|------|----------|
| **딜러직** | 채용 인원의 대다수(36/47명), 교대근무수당 |
| 사무행정직/마케터직 | 소수 채용, 일반 행정직 기본급 |
| 전기/안전관리직 | 기술수당 별도 |

## 4. 성과보수표 (경영평가 성과급)

### 4.1 공기업 직원 경영평가성과급

| 등급 | 지급률(기준월봉 대비) |
|------|---------------------|
| **S (탁월)** | **250%** |
| **A (우수)** | **200%** |
| **B (양호)** | **150%** |
| **C (보통)** | **100%** |
| **D (미흡)** | **0% (미지급)** |
| **E (아주 미흡)** | **0% (미지급)** |

### 4.2 준정부기관 직원 경영평가성과급

| 등급 | 지급률(기준월봉 대비) |
|------|---------------------|
| **S (탁월)** | **100% + 20% 차등** |
| **A (우수)** | **100%** |
| **B (양호)** | **80%** |
| **C (보통)** | **차등 지급 (대폭 감소)** |
| **D (미흡)** | **0% (미지급)** |
| **E (아주 미흡)** | **0% (미지급)** |

### 4.3 내부평가급

- **기준월봉의 100%** (공기업/준정부기관 예산운용지침 기준)
- 기관 자체 성과평가 등급에 따라 S/A/B/C 개인별 차등 지급

### 4.4 보수 구성비 예시 (한국관광공사)

| 항목 | 비율 |
|------|------|
| 기본급 | **72%** |
| 수당 | **6%** |
| 성과급(경영평가+내부평가) | **20%** |
| 기타 | **2%** |

(출처: 기획재정부 성과급지급기준; ALIO)

## 6. 복리후생 특이사항

### 공통 복리후생
- 4대 보험, 퇴직연금(DB/DC형)
- 연차휴가 + 하계휴가(3~5일)
- 선택적 복지포인트: 연 100~200만원
- 건강검진(본인+배우자), 경조사 지원
- 자기개발비/교육훈련비: 연 50~200만원

### 기관별 특이 복리후생

| 기관 | 특이 혜택 |
|------|----------|
| **한국관광공사** | 해외사무소(약 30개국) 근무 기회, 해외근무수당 별도 |
| **EBS** | 방송사 특성상 제작수당/야간수당 별도 |
| **세종학당재단** | 해외 세종학당 파견, 어학 관련 수당 |

| **그랜드코리아레저** | 교대근무수당, 딜러직 전문교육 제공 |
