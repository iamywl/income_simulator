## 1. 채용 및 직급 체계

### 1.1 직급 구조

일반직 공무원은 **1급(관리관)~9급(서기보)** 의 9단계 직급 체계로 구성된다 (「국가공무원법」 제4조; 인사혁신처, 2025).

| 직급 | 직위명 | 비고 |
|------|--------|------|
| 1급 | 관리관 | 고위공무원단 |
| 2급 | 이사관 | 고위공무원단 |
| 3급 | 부이사관 | 고위공무원단 |
| 4급 | 서기관 | 중간관리자 |
| 5급 | 사무관 | 실무 관리자 |
| 6급 | 주사 | 실무자 |
| 7급 | 주사보 | 실무자 |
| 8급 | 서기 | 실무자 |
| 9급 | 서기보 | 초급 실무자 |

(인사혁신처, 「공무원 인사 실무」, 2025)

### 1.2 채용시험 종류

- **국가직 공무원**: 인사혁신처 주관, 5급(행정고시/외교관후보자선발시험), 7급, 9급 공개경쟁채용시험 (인사혁신처, 2025)
- **지방직 공무원**: 각 시·도 인사위원회 주관, 7급·9급 공개경쟁채용시험 (각 시·도 인사위원회, 2025)
- **서울시 공무원**: 서울특별시 인사위원회 주관, 별도 채용공고 (서울특별시, 2025)
- **경력경쟁채용**: 특정 자격·경력 보유자 대상 (「국가공무원법」 제28조)

취업준비생이 주로 응시하는 시험은 **9급 공채**(고졸 이상)와 **7급 공채**(학력 무관, 사실상 대졸 수준)이며, 5급 공채(행정고시)는 고시 준비생이 도전한다.

## 5. 복리후생

### 5.1 공무원연금

- **기여율**: 본인 봉급의 **9%** (기관 9% 매칭, 총 18%)
- **수급요건**: 10년 이상 재직 후 65세 도달 시 연금 수령
- **연금액**: 재직기간 × 1.7% × 평균기준소득월액 (2025년 기준)
- 30년 재직 시 평균소득의 약 **51%** 수준 연금 수령 가능
- (「공무원연금법」 제47조, 공무원연금공단, 2025)

### 5.2 건강보험

- 본인 부담률: 보수월액의 **3.545%** (2025년 기준)
- 장기요양보험료 포함 시 실질 부담률 약 **3.99%**
- (국민건강보험공단, 2025)

### 5.3 복지포인트

- 연간 **400,000~1,200,000원** 수준 (기관·직급·근속연수별 상이)
- 기본 포인트(약 40~60만원) + 근속 가산 포인트
- 자기계발, 건강관리, 여가활동, 생활용품 구매 등에 사용
- (각 기관 복지운영규정, 2025)

### 5.4 주택자금대출 (공무원연금공단)

- **주택구입대출**: 최대 **7,000만원**, 금리 연 2.0~2.5% 수준 (시중은행 대비 저리)
- **전세자금대출**: 최대 **4,000만원**
- 재직기간 5년 이상 시 신청 가능 (일부 상품은 1년 이상)
- (공무원연금공단, 2025)

### 5.5 자녀학비보조수당

- **고등학교**: 입학금 + 수업료 전액 지원
- **대학교**: 지원 폐지됨 (2015년 이후 단계적 폐지 완료)
- (「공무원수당 등에 관한 규정」 제25조)

### 5.6 기타 복리후생

| 항목 | 내용 |
|------|------|
| 연가 | 재직 1년 미만 11일, 이후 1년마다 1일 가산, 최대 21일 |
| 병가 | 연 60일 (유급), 공무상 질병은 연 180일 |
| 육아휴직 | 최대 3년 (첫 1년 봉급의 80%, 이후 무급) |
| 육아시간 | 만 8세 이하 자녀, 일 2시간 단축근무 |
| 공무원 맞춤형 건강검진 | 격년 1회 무료 건강검진 |
| 공무원 할인 | 국립시설, 공공기관 할인 혜택 |
| 공무원 단체보험 | 기관별 단체보험 가입 (저렴한 보험료) |

(인사혁신처, 「공무원 복무규정」, 2025; 공무원연금공단, 2025)

## 6. 승진 소요 연수

### 6.1 법정 최소 승진 소요 연수

| 승진 구간 | 최소 소요 연수 |
|-----------|---------------|
| 9급 → 8급 | 1년 6개월 |
| 8급 → 7급 | 2년 |
| 7급 → 6급 | 3년 6개월 |
| 6급 → 5급 | 4년 |
| 5급 → 4급 | 3년 |

(「공무원임용령」 제31조, 2025)

### 6.2 실제 평균 승진 소요 연수

법정 최소와 실제 소요기간은 크게 다르다. 기관, 직렬, 경쟁률에 따라 상이하나 일반적 평균은 다음과 같다.

| 승진 구간 | 실제 평균 소요 연수 | 누적 연수 |
|-----------|-------------------|-----------|
| 9급 → 8급 | 3~5년 | 3~5년 |
| 8급 → 7급 | 4~6년 | 7~11년 |
| 7급 → 6급 | 5~8년 | 12~19년 |
| 6급 → 5급 | 8~12년 | 20~31년 |

(인사혁신처, 「공무원 인사통계」, 2024; 각 기관 인사자료 종합)

> **취업준비생 참고사항**:
> - 9급 입직 시 5급(사무관) 승진까지 평균 **20~30년** 소요
> - 7급 입직 시 5급 승진까지 평균 **13~20년** 소요
> - 5급(행정고시) 입직 시 바로 사무관으로 시작
> - 승진 속도는 기관 규모, T/O, 성과평가, 근무평정 등에 크게 좌우됨
