# 한국 교원(유치원/초등/중등/고등학교 교사) 임금 상세 조사 보고서

> 취업준비생 관점에서 작성 | 2025년 기준

---

## 1. 채용 및 직급 체계

### 1.1 교원의 직급 체계

한국의 교원 직급 체계는 **교사 -> 교감 -> 교장**의 3단계로 구성된다 (교육공무원법 제2조).

| 직급 | 역할 | 비고 |
|------|------|------|
| **교사** | 학생 교육 직접 담당 | 정교사 2급 -> 1급 승급 |
| **교감** | 교장 보좌, 학교 운영 관리 | 교육경력 + 자격연수 필요 |
| **교장** | 학교 최고 관리자 | 교감 경력 + 자격연수 또는 공모교장제 |

- **교감 승진 요건**: 교사 또는 장학사/교육연구사로서 소정의 교육경력을 갖추고 자격연수를 이수한 뒤 승진 심사를 통과해야 한다.
- **교장 임용**: 교감 경력을 바탕으로 교장 자격을 취득하거나, **공모교장제**를 통해 임용될 수 있다.

### 1.2 임용시험 종류

법정 명칭은 **"교육공무원 임용후보자 선정경쟁시험"**이다 (위키백과, "대한민국의 교사임용시험," 2025).

| 시험 종류 | 대상 | 응시 자격 |
|-----------|------|-----------|
| **초등교원임용시험** | 유치원/초등학교 교사 | 교육대학교(교대) 졸업자, 초등교육과 졸업자 |
| **중등교원임용시험** | 중학교/고등학교 교사 | 사범대학 졸업자, 교직이수자, 교육대학원 수료자 |

- 초등교원임용시험은 각 시도교육청 단위로 시행
- 중등교원임용시험은 전국 단위로 과목별 선발
- 시험은 1차(필기) -> 2차(수업실연, 면접, 실기 등)로 진행

## 3. 수당 체계 상세

### 3.1 정근수당

근무연수에 따라 **연 2회**(1월, 7월) 지급된다. 2025년부터 **1년 미만 교원에게도 10% 지급**으로 확대되었다 (에듀프레스, 2024.12).

| 근무연수 | 지급률(월봉급액 대비) |
|----------|---------------------|
| 1년 미만 | 10% (2025년 신설) |
| 1년 이상 ~ 2년 미만 | 15% (기존 10%에서 인상) |
| 2년 이상 ~ 3년 미만 | 20% (기존 15%에서 인상) |
| 3년 이상 ~ 4년 미만 | 25% (기존 20%에서 인상) |
| 4년 이상 ~ 5년 미만 | 30% (기존 25%에서 인상) |
| 5년 이상 ~ 6년 미만 | 35% |
| 6년 이상 ~ 7년 미만 | 40% |
| 7년 이상 | 50% |

### 3.2 정근수당 가산금

매월 지급되며, 2025년부터 **5년 미만 교원에게도 월 3만원 지급**으로 확대되었다 (한국교원신문, 2025).

| 근무연수 | 월 지급액 |
|----------|----------|
| 5년 미만 | 30,000원 (2025년 신설) |
| 5년 이상 ~ 10년 미만 | 50,000원 |
| 10년 이상 ~ 15년 미만 | 60,000원 |
| 15년 이상 ~ 20년 미만 | 80,000원 |
| 20년 이상 | 100,000원 |

## 5. 복리후생

### 5.1 공무원연금

- **기여금 비율**: 기준소득월액의 **9%** (2020년 이후 고정)
- **기준소득월액**: 기본급뿐 아니라 정근수당, 성과상여금 등 과세소득 전체를 12개월 평균한 금액
- **납부 기간 상한**: 최대 **36년** (이후 기여금 납부 면제)
- **수령 조건**: 재직기간 10년 이상이면 퇴직 후 연금 수급 가능, 65세부터 수령
- **연금 산정**: 재직기간 1년당 기준소득의 약 1.7%

(출처: 공무원연금법; 공무원연금공단, geps.or.kr)

### 5.2 건강보험

- 본인부담률: 보수월액의 약 **3.545%** (2025년 기준, 사용자 동일 부담)

### 5.3 복지포인트 (맞춤형 복지제도)

- **기본점수**: 연간 약 400~500점
- **근속점수**: 근무연수 x 10점
- **가족점수**: 부양가족 1인당 연 30~50점
- 1포인트 = 1,000원, 연간 총 약 **70만~120만원** 상당
- 건강검진비 격년 **별도 20만원** 추가 지원

### 5.4 자녀학비보조수당

- **중학생/고등학생 자녀**: 입학금, 수업료, 학교운영지원비 실비 지원
- 공립학교는 해당 없고 사립학교 재학 시 실비 보조
- 대학생 자녀: **학비보조 제도 폐지됨**

### 5.5 기타 복리후생

| 항목 | 내용 |
|------|------|
| **안식년(학습연구년)** | 10년 이상 근무 시 1년간 연구/학습 기회 (일부 시도교육청) |
| **육아휴직** | 자녀 만 8세 이하, 최대 3년 (첫 3개월 봉급 80%) |
| **공무원 대출** | 공무원연금공단 저리 대출 (주택/생활안정자금) |
| **연가** | 근무연수에 따라 연 11~21일 |
| **공무상 요양** | 공무 중 부상/질병 시 치료비 전액 지원 |

## 6. 호봉 획정: 학력별 초임 호봉 결정 방법

교원의 초임 호봉은 단순히 1호봉에서 시작하는 것이 아니라, **학력, 경력, 자격** 등을 반영하여 획정된다 (교육부 예규 제97호).

### 6.1 초임호봉 계산 공식

```
초임호봉 = 기산호봉 + (학령 - 16) + 가감연수 + 환산경력
```

### 6.2 기산호봉

| 자격 | 기산호봉 |
|------|---------|
| 정교사 2급 (4년제 대학 졸업) | 8호봉 |
| 정교사 2급 (교대/사대 졸업) | 8호봉 |
| 준교사 | 7호봉 |

### 6.3 학력별 전형적인 초임 호봉

| 학력/경로 | 초임 호봉 | 비고 |
|-----------|----------|------|
| 교대 졸업 (4년) | **9호봉** | 기산 8호봉 + 교생실습 등 +1 |
| 사범대 졸업 (4년) | **9호봉** | 동일 |
| 일반대 + 교직이수 (4년) | **8호봉** | 교대/사대 가산 없음 |
| 교육대학원 졸업 (석사 2년) | **10호봉** | 대학원 2년 경력환산 100% 반영 |
| 사범대 + 석사 (6년) | **11호봉** | 석사 2년 경력 가산 |
| 군 복무 경력 (남성) | **+1~2호봉** | 군 경력 100% 환산 |
| 기간제 교사 경력 | **+경력연수** | 기간제 경력 100% 환산 |
| 사설학원 강사 경력 | **+경력연수x30%** | 환산율 30% 적용 |

### 6.4 경력환산율

| 경력 유형 | 환산율 |
|-----------|--------|
| 교육기관 교원 경력 (기간제 포함) | 100% |
| 군 복무 | 100% |
| 국가/지방공무원 경력 | 100% |
| 관련 분야 민간기업 경력 | 80% |
| 비관련 분야 민간기업 경력 | 50% |
| 사설학원 강사 경력 | 30% |

(출처: 교육부 예규 제97호, "교육공무원 호봉획정시 경력환산율표의 적용 등에 관한 예규")
