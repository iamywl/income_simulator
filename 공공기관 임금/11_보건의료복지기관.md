## 2. 주요 기관 상세

### 2.1 국민건강보험공단 (NHIS)

**직급 체계**

| 직급 | 직위 | 비고 |
|------|------|------|
| 1급 | 지사장급 | 고위관리직 |
| 2급 | 부장 | |
| 3급 | 팀장 | |
| 4급 | 과장/차장 | |
| 5급 | 대리 | |
| 6급 | 주임 (신입) | 대졸 신규채용 진입직급 |

**보수 구조 (보수규정 기준)**
- **기본연봉**: 연봉기준급의 **84%** (매월 균등 분할 지급)
- **성과연봉**: 연봉기준급의 **16%** (전년도 업무실적 평가결과 반영)

**직무별 구분**: 행정직, 건강직(보건의료 면허소지자), 전산직. 직무별 기본급 차이는 크지 않으나 수당 체계에서 차이 발생.

### 2.2 건강보험심사평가원 (HIRA)

**직급 체계**: 1급~5급 (행정직, **심사직**, 전산직, 연구직 4개 직종)

**심사직 상세 (핵심 직군)**

| 구분 | 채용 직급 | 자격요건 |
|------|-----------|----------|
| 심사직 (약사) | **4급** | 약사 면허 + 관련 경력 1년 이상 또는 약학 석사 |
| 심사직 (간호사/의료기사) | **5급** | 면허 + 종합병원급 이상 임상/심사 경력 1년 이상 |
| 심사직 (의사) | **3급 이상** | 의사 면허 + 임상 경력 |

- 심사직은 **간호사가 절대 다수**
- 약사는 4급으로 채용되어 간호사(5급)보다 1단계 높은 직급 시작
- 심사직은 행정직 대비 **전문수당(의료면허수당 등)** 추가 지급

**연봉 정보**
- 신입 초봉: 약 4,125만원
- 7년차 예상: 약 4,636만원
- 8년차 예상: 약 4,683만원

## 3. 성과보수표

### 3.1 준정부기관 직원 경영평가성과급

| 등급 | 지급률(기준월봉 대비) |
|------|---------------------|
| S (탁월) | 100% + 20% 차등 |
| A (우수) | 100% |
| B (양호) | ~80% |
| C (보통) | ~60% |
| D (미흡) | 미지급 |
| E (아주미흡) | 미지급 |

### 3.2 내부평가급

- **기준월봉의 100%** (기관 내부 평가에 따라 개인별 차등)
- 산정공식: `기준월봉 x 성과급지급률 x 근무계수 x 조정계수`

### 3.3 건보공단 보수 구조

| 구분 | 비율 | 내용 |
|------|------|------|
| 기본연봉 | 84% | 매월 균등 분할 지급 |
| 성과연봉 | 16% | 전년도 업무실적 평가 반영 |

(출처: NHIS 보수규정; 기획재정부 성과급지급기준)
