## 2. 공통 직급체계

| 직급 | 직책 대응 | 비고 |
|------|----------|------|
| **1급** | 본부장, 실장 | 임원 바로 아래 |
| **2급** | 처장, 부장 | |
| **3급** | 팀장, 차장 | 고과 승진 시작 |
| **4급** | 과장, 선임 | 자동 진급 기관 多 |
| **5급** | 대리, 주임 | 대졸 신입 입사 |
| **6급** | 사원 | 고졸/전문대 신입 |

- 한국농어촌공사: 5급·6급 입사, **4급까지 자동진급**, 3급부터 고과 승진
- 국가철도공단: 주임(5급) → 과장(4급) → 팀장(3급) → 처장(2급) → 본부장(1급)

## 3. 직무별 연봉 차이

| 직무 유형 | 특징 | 연봉 차이 요인 |
|-----------|------|---------------|
| **사무직(행정)** | 일반 행정, 기획, 회계 | 기본급 기준, 별도 수당 적음 |
| **기술직** | 토목, 전기, 환경, IT | 기술수당, 자격수당 추가 (월 5~20만원) |
| **연구직** | 박사급 연구원 | 별도 연봉테이블, 초봉 +30~50% |
| **현장직** | 시설관리, 현장점검 | 위험수당, 특수근무수당 추가 |

- **항만공사**: 항만운영직, 기계·전기직에 교대근무수당·항만특수수당 추가
- **국립공원공단, 한국농어촌공사, LX**: 위험·오지 근무수당, 야간수당 추가

## 4. 성과보수표

### 4.1 공기업 직원 성과급 (월 기본급 대비)

| 평가등급 | 지급률 |
|----------|--------|
| S등급 | **250%** |
| A등급 | **200%** |
| B등급 | **150%** |
| C등급 | **100%** |
| D/E등급 | **0%** |

### 4.2 준정부기관 직원 성과급 (월 기본급 대비)

| 평가등급 | 지급률 |
|----------|--------|
| S등급 | 약 **200%** |
| A등급 | 약 **150%** |
| B등급 | 약 **100%** |
| C등급 | 약 **50%** |
| D/E등급 | **0%** |

### 4.3 성과급 산정 공식

```
직원 성과급 = 기준월봉 x 성과급지급률 x 근무계수 x 조정계수
```

(출처: 기획재정부 2025년도 성과급지급기준)
