## 3. 보수 체계

### 3.1 호봉제에서 직무급제로의 전환

- **2024년 기준**: 직무급 도입 공공기관 **129곳** (전년 108곳 대비 증가)
- **공기업/준정부기관 도입률**: **93.1%** (81개 기관)
- **정부 목표**: 2027년까지 모든 공공기관(200곳)에 직무급 도입, 호봉제 완전 폐지 추진
- 직무급 도입 기관에는 경영실적 평가 시 **가점 부여** 및 **총인건비 추가 인상** 인센티브 제공

(기획재정부, 2023; 정책브리핑, 2025.04)

### 3.2 현행 보수 구조

대부분의 공기업은 아래와 같은 **혼합형 연봉제** 구조를 운영한다:

```
총보수 = 기본급(연봉) + 성과급 + 제수당 + 복리후생
```

| 구분 | 내용 |
|------|------|
| **기본연봉** | 직급/직무등급별 연봉 테이블에 따라 책정 |
| **성과연봉** | 전년도 업적평가 결과에 연동. 기본급의 0~20% 수준 차등 |
| **경영평가 성과급** | 기획재정부 경영평가 등급에 따라 별도 지급 |
| **제수당** | 시간외근무수당, 야간/휴일수당, 직책수당, 자격수당 등 |

## 4. 수당 및 성과급 상세

### 4.1 경영평가 성과급

| 평가등급 | 직원 성과급 지급률 (월 기본급 대비) |
|----------|--------------------------------------|
| **S등급** (탁월) | 250% |
| **A등급** (우수) | 200% |
| **B등급** (양호) | 150% |
| **C등급** (보통) | 100% |
| **D등급** (미흡) | 미지급 |
| **E등급** (아주 미흡) | 미지급 |

- 신입사원의 경우, 월 기본급이 약 250~300만 원 수준이므로 S등급 기관 재직 시 약 **625~750만 원**의 경영평가 성과급 수령 가능
- 2024년 경영평가에서 한전, 한수원 등 에너지 공기업 5개사가 대거 'A' 등급 달성 (팍스경제TV, 2025)

(출처: 기획재정부, 「공기업/준정부기관 예산운영지침」)

### 4.2 내부 성과급

- 기관 자체 업적/역량 평가에 기반하여 지급
- 통상 기본급의 **0~200%** 범위 내에서 차등 지급

## 취업준비생을 위한 핵심 시사점

1. **초봉 vs 평균연봉의 괴리**: 인천공항은 초봉 1위(4,894만)이나 평균연봉은 중위권. 반면 한국가스공사는 초봉 2위이면서 평균연봉도 1위(9,602만)로 장기적 보상이 뛰어남.

2. **교대근무 기관의 실수령액**: 발전 5사(남동/중부/서부/남부/동서), 한수원, 코레일 등은 교대근무수당, 야간수당이 크므로 기본 초봉 대비 실수령이 크게 높아짐.

3. **경영평가 등급의 중요성**: S등급 기관은 월 기본급의 250%를 성과급으로 수령. 신입 기준 약 625~750만 원 추가 수입. D/E등급은 성과급 없음.

4. **직무급 전환 영향**: 2024년 기준 93.1%의 공기업이 직무급을 도입하여, 향후 연공서열에 따른 자동 급여 인상 폭이 줄어들 수 있으나, 직무 전문성에 따른 보상 차등화 가능.

5. **LH/코레일의 상대적 저연봉**: LH(3,507만), 코레일(약 3,500만) 등은 초봉이 하위권이나, 고용안정성과 복리후생(사택, 교통 할인 등)을 종합적으로 고려해야 함.
