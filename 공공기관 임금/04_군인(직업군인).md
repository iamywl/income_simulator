## 1. 채용 및 계급 체계

### 1.1 계급 구조

| 구분 | 계급 |
|------|------|
| **병** | 이등병 → 일등병 → 상등병 → 병장 |
| **부사관** | 하사 → 중사 → 상사 → 원사 |
| **준사관** | 준위 |
| **장교(위관)** | 소위 → 중위 → 대위 |
| **장교(영관)** | 소령 → 중령 → 대령 |
| **장교(장관)** | 준장 → 소장 → 중장 → 대장 |

(위키백과, "군사 계급"; 「군인사법」)

### 1.2 임관 경로
