# 한국 공기업 직급별/직무별 연봉 및 성과보수표 상세

> 06_공기업.md 보강 자료 | 2024~2025년 기준

---

## 1. 공기업 공통 직급 체계

### 1.1 일반적 직급 구조

| 직급 | 직위 호칭 | 대졸 기준 도달 연차 | 추정 연봉 범위(만원) |
|------|----------|-------------------|---------------------|
| 6급(신입) | 사원/주임 | 입사 시 | 3,500~4,500 |
| 5급 | 대리 | 입사 후 3~5년 | 4,500~5,800 |
| 4급(을/나) | 과장 | 입사 후 7~10년 | 5,800~7,500 |
| 4급(갑/가) | 차장 | 입사 후 12~17년 | 7,500~9,000 |
| 3급 | 부장 | 입사 후 17~22년 | 9,000~11,000 |
| 2급 | 처장/실장 | 입사 후 22년 이상 | 11,000~13,000 |
| 1급 | 본부장/이사 | 임원급 | 13,000 이상 |

### 1.2 연봉 구성 구조

```
총보수 = 기준급(호봉) + 직무급 + 정기상여금(400~600%) + 경영평가 성과급 + 내부 성과급 + 제수당
```

## 2. 직무별 차이

### 2.1 직무별 초봉: 동일 직급 동일 초봉 원칙

| 직무 | 초봉 기본급 | 실수령 차이 요인 |
|------|-----------|-----------------|
| **사무직** (경영/기획/인사/재무) | 동일 | 시간외근무수당 상대적 적음, 본사 근무 비율 높음 |
| **기술직** (전기/기계/토목/건축/화학) | 동일 | 현장수당, 위험근무수당, 교대근무수당 추가 → **연간 +500~1,000만원** |
| **전산직** (IT/시스템운영) | 동일 | 사무직과 유사, 일부 기관 IT전문가 수당 별도 |
| **연구직** | 동일 또는 별도 | 학위수당(석사/박사) 추가 |

## 3. 주요 기관별 직급 체계 상세

### 3.1 한국전력공사 (한전)

**2024년 12월 직급체계 개편**:
- 기존 '4(갑)-4(을)-5직급-6직급' → '**선임-일반-현장/기술-전문/사무**' 체계로 변경
- 호칭: 입사 직후 '주임' → 2년 후 '대리' → 이후 8년 근무 시 '과장'

**기본급 범위**:

| 직급 | 기준급 범위 | 상한 기본급 |
|------|-----------|-----------|
| 4직급 | 3,350만~8,726만원 | 8,446만원 |
| 5직급 | - | 8,172만원 |
| 6직급 | - | 7,578만원 |
| 대졸 초임 | 기준급 3,760만원 | 직무급 월 14만원 |

- 직무급: 월 14~34만원 (직급별 차등)

(출처: 한국경제, 2024.12.24; 뉴스1)

### 3.2 한국가스공사

- 직급: 4급(신입) → 3급 → 2급 → 1급
- 초봉: 약 4,796~4,800만원
- 급여 구성: 기본급 + **연고정성과 550%** + 경영평가 성과급(0~250%)
- 7년차 약 6,359만원, 8년차 약 7,193만원

### 3.3 한국철도공사 (코레일)

- 직급: 5급(신입) → 4급 → 3급 → 2급 → 1급
- 일반직 5급 초임: 약 3,400만원 / 운전직 초임: 약 3,700만원
- 철도기관사 연봉 범위: 3,500만원(초봉) ~ 1억원(정년 퇴직 전)
- **직렬별 차이**: 운전직/차량직은 야간/휴일 수당으로 사무직 대비 실수령 높음

### 3.4 LH (한국토지주택공사)

- 직급: 5급(대졸)/6급(고졸) → 4급 → 3급 → 2급 → 1급
- 5급 사무직: 일반행정/법률/회계/전산
- 5급 기술직: 토목군(토목/도시계획/조경/환경/교통), 건축군(건축/기계/전기)
- 초봉: 약 3,507만원 (공기업 중 하위권)
- 평균연봉: 약 6,542~7,200만원

### 3.5 인천국제공항공사

- 초봉: **4,894만원** (공기업 초봉 1위)
- 평균연봉: 약 8,700~9,765만원
- 과장급: 약 4,900만원 / 임원: 약 12,371만원

### 3.6 한국수자원공사 (K-water)

- 초봉: 약 4,061~4,100만원
- 평균연봉: 약 8,194~8,353만원
- 과장급: 약 5,833만원 / 임원: 약 10,819만원
- 7년차: 약 5,216만원, 8년차: 약 5,616만원

## 4. 성과보수표 상세

### 4.1 경영평가 성과급 (기관 단위)

**공기업 직원 성과급** (월 기본급 대비):

| 평가등급 | 지급률 | 신입(월기본급 300만원 가정) 수령액 |
|----------|--------|--------------------------------|
| **S등급** (탁월) | **250%** | 약 750만원 |
| **A등급** (우수) | **200%** | 약 600만원 |
| **B등급** (양호) | **150%** | 약 450만원 |
| **C등급** (보통) | **100%** | 약 300만원 |
| **D등급** (미흡) | **0%** | 0원 |
| **E등급** (아주미흡) | **0%** | 0원 |

**준정부기관 직원 성과급** (월 기본급 대비):

| 평가등급 | 지급률 |
|----------|--------|
| S등급 | **100%** |
| A등급 | **80%** |
| B등급 | **60%** |
| C등급 | **40%** |
| D/E등급 | 0% |

> 공기업이 준정부기관보다 성과급 지급률이 현저히 높음 (최대 250% vs 100%)

### 4.2 내부 개인 성과급 차등 배분

| 개인 등급 | 경영평가성과급 지급률 | 내부평가급 지급률 |
|-----------|---------------------|-----------------|
| S등급 | 93.4% | 134% |
| A등급 | 81.7% | 117% |
| B등급 | 70.0% | 100% |
| C등급 | 58.3% | 83% |
| D등급 | 46.6% | 66% |

- B등급(보통)을 100% 기준으로 상위 가산, 하위 감산
- 등급 간 차등폭: 11.7%p (대칭 차등)

(출처: ALIO 2025년도 성과급지급기준)

### 4.3 정기상여금

- 기본급의 **400~600%** (연간, 기관별 상이)
- 한국가스공사: **연고정성과 550%**
- 분기별 또는 격월로 분할 지급

## 5. 취업준비생 핵심 요약

1. **직무별 초봉 차이는 기본급 기준으로 없음**. 기술직/운전직은 수당으로 연 500~1,000만원 추가.
2. **경영평가 등급이 연봉에 미치는 영향이 극대**: S등급 약 750만원 vs D등급 0원.
3. **한전 직급체계 2024년 12월 대폭 개편**: 숫자직급 → 직무중심 체계.
4. **공기업 vs 준정부기관 성과급 차이**: 최대 250% vs 최대 100%.
5. **정기상여금 400~600%**가 기본급에 추가되므로 초봉 표기액보다 실수입이 상당히 높음.
